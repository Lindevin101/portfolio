# My Portfolio Website
Live Link: https://main--mellow-biscochitos-b7290b.netlify.app/

This is a personal portfolio website built using React. It showcases my skills, projects, and professional details.

## Features

- Responsive design with a modern and clean UI.
- Smooth scrolling for seamless navigation.
- Sections:
  - **Header**: Contains navigation links and LinkedIn handle.
  - **About**: A brief introduction about myself.
  - **Projects**: Highlights of my work and accomplishments.
  - **Contact**: A section for visitors to get in touch with me.

## Getting Started

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/) (comes with Node.js)
